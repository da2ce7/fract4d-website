Download Information

Gnofract 4D works on modern Unix-based operating systems such as Linux and FreeBSD. An older version has been ported to Mac OS X. It is not currently available for Windows.

If you have any difficulties downloading, installing or compiling, please see the FAQ or let me know.

Source Code

Download this if you want to compile Gnofract 4D yourself, look at how it works, or you use an OS or distribution which doesn't use RPM files. (Or if you don't know what any of that means.) To compile, you will need header files for GTK2 and Python and a C++ compiler. Unpack the package with "tar xzvf gnofract4d-3.1.tar.gz" then see the README file for instructions on how to compile and install.

Binary RPM packages

Download one of these if you have a Linux distribution which uses RPM files running on an Intel or AMD computer. It doesn't need to be compiled, but you still need a compiler installed to run it. You also need Python, PyGTK and GTK+ 2.x.

You need to download the version matching the Python version and processor architecture you're using. If you don't know, use "python -V" to find out your Python version, and "uname -i" to find the processor type.

Install the package using "rpm -i <rpmfile>".

Packages from elsewhere

Other people have kindly repackaged Gnofract 4D for additional Linux distributions and OSes. You may find these useful. However please note I don't maintain these and in most cases have never used them. They are often for an older release than the current one, especially if the new release has only just come out.
