Bugs and Known Issues

Reporting Bugs

Please report any bugs you encounter, either by mailing catenary@users.sourceforge.net or by using the bug form at SourceForge. I'll do my best to fix them.

Backwards Compatibility

Version 2.x contains major architectural changes from version 1.x, so unfortunately not all images generated by earlier versions will look exactly the same or even load correctly. If you have a favorite image which doesn't work any more, mail me the .fct file and I'll try and convert it. I hope to fix most of these issues in future versions.

Specific known issues:

  • Files generated by Gnofract 4D versions older than 1.4 can't be loaded.

  • Only one colormap per file is supported (the outer one). Inner colormaps are ignored.
